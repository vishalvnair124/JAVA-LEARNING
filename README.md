## 🛠️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/vishalvnair124/JAVA-LEARNING.git
```

### Navigate to Any Project

```bash
cd JAVA-LEARNING/<Project-Folder>
```

### Compile

```bash
javac Main.java
```

### Run

```bash
java Main
```
